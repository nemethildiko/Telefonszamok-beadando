# Telefonszámok – WPF alkalmazás
## Leírás
Egyszerű telefonkönyv alkalmazás WPF + Entity Framework Core használatával.
Személyek és hozzájuk tartozó telefonszámok kezelése.
## Funkciók
- Személyek listázása
- Új személy felvétele
- Személy adatainak módosítása
- Telefonszám hozzáadása
- Telefonszám törlése
- Helység kiválasztása ComboBoxból
## Adatbázis
- SQL Server (LocalDB)
- Adatbázis neve: Telefonszamok
## Bejelentkezési adatok
Felhasználónév: admin
Jelszó: admin
